# FilmDev — iPhoneで試す手順

このフォルダ（`filmdev`）は、そのまま無料ホストに置けばiPhoneに「アプリ」として入れられます。
中身：`index.html` / `manifest.webmanifest` / `sw.js` / `icon-180.png` / `icon-192.png` / `icon-512.png`

> ⚠️ カメラ撮影は **HTTPS** でないと動きません。下の手順で公開すれば自動でHTTPSになります。
> （ファイルを直接開くと「写真を選ぶ」だけ動作します）

---

## いちばん簡単：Netlify Drop（アカウント不要・ドラッグ&ドロップ・数分）

1. PC（Mac/Win）のブラウザで **https://app.netlify.com/drop** を開く
2. この **`filmdev` フォルダごと** 画面にドラッグ&ドロップ
3. `https://xxxx.netlify.app` のような **https URL** が発行される
4. そのURLをiPhoneのSafariで開く → 下の「ホーム画面に追加」へ

※ URLを自分だけのメモにしておけば、次回もそこから開けます。

---

## 代替：GitHub Pages（GitHubアカウントがある人向け）

1. GitHubで新しいリポジトリを作成（例：`filmdev`、Public）
2. `filmdev` フォルダの中身（index.html など）をすべてアップロード
   （「Add file → Upload files」でドラッグ&ドロップ）
3. リポジトリの **Settings → Pages** を開く
4. **Source** を `Deploy from a branch`、Branch を `main` / `(root)` にして Save
5. 数十秒後に `https://ユーザー名.github.io/filmdev/` が発行される
6. そのURLをiPhoneのSafariで開く → 下の「ホーム画面に追加」へ

---

## iPhoneでホーム画面に追加（PWAインストール）

1. **Safari**（Chrome不可。iOSはSafariのみPWA対応）で公開URLを開く
2. 下部中央の **共有ボタン**（□に↑）をタップ
3. メニューを下にスクロールして **「ホーム画面に追加」** をタップ
4. 名前（FilmDev）を確認して **「追加」**
5. ホーム画面にFilmDevアイコンが出る → タップすると全画面のアプリとして起動

初回だけオンラインで開けば、以降はオフラインでも起動できます（現像処理は端末内で完結）。

---

## 使い方

1. **📷 カメラ（高解像度）** でネガを撮影（iPhoneの標準カメラが開き、フル解像度で取り込み）／または **🖼 写真を選ぶ**
2. トリミング画面でコマに枠を合わせる（**✨ 自動トリミング**あり）。色が変なら **🎯 ベース指定** でコマ外のネガ地色をタップ
3. **フィルムタイプ**（カラーネガ / モノクロ / ポジ）を選択
4. **✨ 自動補正** → 必要ならカラーバランス・露出・コントラスト・彩度で微調整
5. **⬇ 保存** で写真として書き出し（保存時はフル解像度で再現像されます）

## スキャン光源のおすすめ（代用品）

専用ライトボックスがなくてもOK。上から順におすすめ：

1. **LEDトレース台（A5）** — 2千円前後。均一光・薄型でいちばん安定。「トレース台」「ライトパッド」で検索
2. **iPad/PCの白画面＋コピー用紙1枚** — 画面の上に白紙を1枚置くと画素のモアレが消える。画面輝度は最大に
3. **明るい窓＋トレーシングペーパー** — 直射日光は避け、曇りの日か北向きの窓で

コツ：部屋は暗めに（映り込み防止）、ネガは光源から数cm浮かせ、スマホはネガにピントを合わせてなるべく大きく写す。
